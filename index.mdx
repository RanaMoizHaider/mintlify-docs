The PHP AI SDK is a framework-agnostic library that brings the full breadth of modern AI capabilities to PHP 8.3+ applications. Whether you're building a chatbot, generating images, running voice sessions in real time, or extracting structured data from documents, the SDK provides one consistent interface that works across providers.

## Get started in minutes

<Steps>
  <Step title="Install the core package">
    ```bash
    composer require aisdk/core aisdk/openai
    ```
  </Step>
  <Step title="Set your API key">
    ```bash
    export OPENAI_API_KEY=sk-...
    ```
  </Step>
  <Step title="Generate your first response">
    ```php
    use AiSdk\Generate;
    use AiSdk\OpenAI;

    OpenAI::create();

    $result = Generate::text('Explain what a closure is in PHP.')
        ->model(OpenAI::model('gpt-4o-mini'))
        ->run();

    echo $result->text;
    ```
  </Step>
</Steps>

<Note>
  The PHP AI SDK requires PHP 8.3 or higher. It is fully framework-agnostic and works in Laravel, Symfony, or plain PHP scripts.
</Note>
